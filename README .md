## Run Locally

Clone the project

```bash
  git clone https://github.com/Soumya-9641/kodagu
```

Go to the project directory

```bash
  first change all environment variable(mongodb url)
  add .env file
  write your mongodb url
  connect that url at db/connection.js file
  cd kodagu
  npm install
  nodemon server.js
```
